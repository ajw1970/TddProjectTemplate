# TddProjectTemplate
Project Template for Starting up TDD with Xunit and FluentAssertions.

This is a C# project template named "TDD Template"

You can build the solution with VS 2017 and then run with F5 to launch a test instance of VS which will allow you to add a new project using the "TDD Template" template. 

To install it in your real instance of VS, browse to the build folder in the ProjectDeployer project and run the .vsix file which is generated on build.

To uninstall it, go to Tools->Extensions and Updates from the VS menu and then uninstall "TDD Startup Project"
